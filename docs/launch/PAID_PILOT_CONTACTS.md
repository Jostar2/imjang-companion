# Paid Pilot Contacts

## Pilot offer default

- format: 2-week founder-led pilot
- target persona: buyer agent first, investor operator second
- target cohort size: 3 to 5 teams
- success outcome:
  - at least 3 completed visits
  - comparison or report opened after a live visit
  - explicit willingness to continue after pilot review

## Target personas

- buyer agent with repeated weekly visits
- small investor operator
- diligence-heavy scout or assistant

## Sourcing lanes

- personal referrals from agents, investors, or assistants already doing repeated visits
- local real-estate operator communities
- founder outbound to people visibly sharing field-visit notes or property review workflows
- past contacts who already use spreadsheets, Notion, or Kakao chat for visit tracking

## Weekly pipeline target

- 15 new outbound messages
- 5 live replies
- 3 workflow review calls
- 1 to 2 active pilot starts

## Qualification checklist

- visits multiple properties per week or month
- already uses spreadsheet, notes app, or photo folders to track visits
- needs to share findings with another decision-maker
- can articulate the cost of losing visit context

## Outbound opener

Use this message for first-touch outreach:

> I am working on a mobile diligence workflow for teams that visit multiple properties and then lose context across notes, photos, and spreadsheets. If you already run repeated visits, I would like to show you a 20-minute workflow review and see whether a 2-week pilot would save report and comparison time.

## Call agenda

1. ask how they capture visit notes today
2. ask what happens between a visit and a recommendation
3. ask who else needs to review the findings
4. show the project -> property -> visit -> comparison/report flow
5. ask whether they would test this on one live route next week

## Exit criteria for this document

This artifact is launch-ready only when:

- at least 10 named leads are listed
- at least 3 calls are scheduled
- at least 1 lead agrees to a live pilot window
