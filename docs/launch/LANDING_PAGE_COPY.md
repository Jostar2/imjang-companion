# Landing Page Copy

## Hero

- Headline: Turn field visits into structured decisions.
- Subheadline: Capture notes, scores, photos, and red flags in one mobile workflow, then compare properties from evidence instead of memory.
- Primary CTA: Request pilot
- Secondary CTA: See workflow

## ICP variant A: Buyer agent / operator

- Promise: Deliver same-day visit reports without stitching together chat, camera roll, and spreadsheets.
- Core proof:
  - mobile-first checklist during the visit
  - comparison across candidate properties
  - report-ready summary after completion

## ICP variant B: Small investor team

- Promise: Standardize diligence notes across repeated visits and keep the rationale behind each recommendation.
- Core proof:
  - structured field capture
  - repeatable checklist categories
  - faster internal review packets

## Objection handling

- "Is this a listing portal?"
  - No. It is a due-diligence workspace after you have real candidates to visit.
- "Does it work offline?"
  - V1 is online-first. It restores in-progress drafts in the same browser, but it does not queue offline writes.
- "Who is it for?"
  - Repeated-use operators first: buyer agents, investor operators, and diligence-heavy property scouts.
